# Funder Research Guide

Systematic approach to researching funders and evaluating fit before applying.

## Overview

Effective funder research saves time by identifying good matches and understanding funder priorities before investing in a full proposal.

## Research Process

### Step 1: Identify Potential Funders

#### Source Types

| Source | Best For | Examples |
|--------|----------|----------|
| Databases | Broad search | Foundation Directory Online, Grants.gov |
| Funder websites | Detailed requirements | Individual foundation sites |
| 990 forms | Past giving patterns | Candid, ProPublica Nonprofit Explorer |
| News/blogs | Current priorities | Foundation newsletters, sector media |
| Networks | Warm introductions | Professional associations, peer orgs |

#### Search Strategies

```
DATABASE SEARCH:
- Geographic focus: [Your location or service area]
- Subject areas: [Your program focus]
- Grant size range: [Appropriate for your request]
- Funding type: [Project, operating, capital]
- Application status: [Currently accepting applications]

990 RESEARCH:
- Look at grants made in past 2-3 years
- Note range of grant sizes
- Identify similar grantees
- Check for patterns in giving
```

### Step 2: Initial Screening

#### Quick Assessment Questions

```
ELIGIBILITY:
[ ] Accepts applications from our organization type?
[ ] Funds in our geographic area?
[ ] Supports our program area/population?
[ ] Grant size appropriate for our request?
[ ] Currently accepting applications?

FIT:
[ ] Mission alignment evident?
[ ] Past grants to similar organizations?
[ ] Grant history suggests ongoing interest?
```

#### Red Flags

- No applications accepted (family foundation with set grantees)
- Geographic restrictions that exclude you
- Funding priorities that do not match your work
- Required relationship/invitation you do not have
- Capacity requirements you cannot meet

### Step 4: Analyze Giving Patterns

#### 990 Form Analysis

```
LOCATE 990:
1. Go to ProPublica Nonprofit Explorer or Candid
2. Search for foundation name
3. Find Schedule F or Part XV (Grants Paid)
4. Download PDF for analysis

WHAT TO LOOK FOR:
- Range of grant sizes (minimum, maximum, typical)
- Types of organizations funded
- Recurring grantees vs. new grantees
- Geographic distribution
- Trends over time (increasing, decreasing, stable)
```

#### Pattern Recognition

| Pattern | Indication |
|---------|------------|
| Same orgs every year | Established relationships prioritized |
| Many new grantees | Open to new applicants |
| Large variations | Strategic shifts occurring |
| All local | Strong geographic preference |
| Few large grants | Focus vs. breadth |

### Step 5: Understand Requirements

#### Application Analysis

```
DOCUMENT:
- Required sections and length limits
- Required attachments
- Budget format and restrictions
- Reporting requirements
- Multi-year funding availability
- Matching/cost share requirements

FORMATTING:
- Font and margin requirements
- Page limits
- File format requirements
- Submission method
```

### Step 6: Build Relationships

#### Outreach Strategies

| Strategy | When Appropriate |
|----------|------------------|
| Information session | Publicly available |
| Program officer call | Explicitly invited in guidelines |
| Email inquiry | Quick clarifying questions |
| Networking event | Professional conferences |
| Peer introduction | Through mutual grantee |

#### Communication Best Practices

```
DO:
- Be concise and respectful of time
- Reference specific programs or priorities
- Ask clarifying questions not answered in guidelines
- Listen more than talk
- Follow up as promised

DO NOT:
- Cold call without invitation
- Ask for funding in first contact
- Ignore stated communication preferences
- Overpromise or overstate capacity
- Contact multiple staff simultaneously
```


## Due Diligence Questions

### Before Applying

```
[ ] Is this funder legitimate? (Verify 501(c)(3) status)
[ ] Are there concerns about funder values or practices?
[ ] Is accepting this funding aligned with our values?
[ ] Can we meet the reporting requirements?
[ ] Is the grant worth the application effort?
[ ] Do we have capacity to implement if funded?
```

## Common Research Mistakes

1. **Insufficient research**: Applying without understanding priorities
2. **Outdated information**: Using old guidelines or priorities
3. **Size mismatch**: Requesting outside typical range
4. **Geographic blind spot**: Missing location restrictions
5. **Relationship assumptions**: Assuming open when invite-only
6. **Deadline errors**: Confusing receipt vs. postmark dates

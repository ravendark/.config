## Search Tips

### Finding Opportunities

```
GRANTS.GOV:
1. Create account
2. Set up saved searches
3. Enable email notifications
4. Check "Forecasted" for upcoming

FOUNDATION RESEARCH:
1. Start with known funders in your area
2. Look at peer organization funders (990s)
3. Use database filters strategically
4. Check funder websites for current priorities
```

### Staying Current

```
RECOMMENDED ROUTINE:
- Weekly: Check new opportunities in databases
- Monthly: Review funder newsletters
- Quarterly: Update prospect tracking
- Annually: Reassess funder landscape
```

## Best Practices

1. **Bookmark key sites**: Create organized bookmark folder
2. **Set up alerts**: Use Grants.gov and Google Alerts
3. **Track sources**: Document where information came from
4. **Verify currency**: Check dates on all guidance
5. **Use primary sources**: Funder websites over aggregators
6. **Build reading routine**: Regular sector news scanning

# Grant Terminology

Key terms and definitions used in grant writing and proposal development.

## Funding Types

| Term | Definition |
|------|------------|
| **Grant** | Funding that does not need to be repaid, awarded for specific purposes |
| **Award** | General term for any funding received |
| **Subgrant** | Grant passed through from one organization to another |
| **Cooperative Agreement** | Grant with substantial funder involvement in activities |
| **Contract** | Funding for specified deliverables with funder oversight |
| **Gift** | Unrestricted funding without specific requirements |

## Budget Terms

| Term | Definition |
|------|------------|
| **Direct Costs** | Expenses directly attributable to the project |
| **Indirect Costs** | Overhead costs not directly attributable (facilities, admin) |
| **F&A** | Facilities and Administration - synonym for indirect costs |
| **IDC Rate** | Indirect Cost Rate - percentage applied to direct costs |
| **Fringe Benefits** | Employee benefits (health, retirement) as percentage of salary |
| **Cost Share** | Portion of project costs borne by applicant |
| **Match** | Synonym for cost share |
| **In-kind** | Non-cash contributions (space, equipment, volunteer time) |
| **Modular Budget** | Simplified budget in standard increments (e.g., $25K modules) |
| **Line Item** | Individual budget category |
| **Subaward** | Portion of funding passed to partner organization |

## Proposal Terms

| Term | Definition |
|------|------------|
| **RFP** | Request for Proposals - funder's call for applications |
| **RFA** | Request for Applications - synonym for RFP |
| **FOA** | Funding Opportunity Announcement |
| **LOI** | Letter of Intent - preliminary expression of interest |
| **Pre-proposal** | Brief initial submission before full proposal |
| **Abstract** | Brief summary (usually 250-500 words) |
| **Specific Aims** | Key objectives (NIH terminology) |
| **Narrative** | Written description of project |
| **Appendix** | Supplementary materials |
| **Biosketch** | Standardized CV format (especially NIH) |

## Evaluation Terms

| Term | Definition |
|------|------------|
| **Logic Model** | Visual representation of program theory |
| **Outcome** | Change resulting from program activities |
| **Output** | Direct product of activities (counts, deliverables) |
| **Impact** | Long-term change at population or systems level |
| **Indicator** | Measurable sign of progress |
| **Baseline** | Starting point for measuring change |
| **Benchmark** | Standard for comparison |
| **Formative Evaluation** | Ongoing evaluation to improve program |
| **Summative Evaluation** | Final evaluation of outcomes |

## Federal Grant Terms

| Term | Definition |
|------|------------|
| **CFDA** | Catalog of Federal Domestic Assistance |
| **SAM** | System for Award Management - federal contractor database |
| **UEI** | Unique Entity Identifier - replaces DUNS number |
| **SF-424** | Standard federal application form |
| **Cognizant Agency** | Federal agency overseeing your indirect cost rate |
| **2 CFR 200** | Uniform Guidance - federal grant regulations |
| **OMB** | Office of Management and Budget |
| **Period of Performance** | Time span when grant activities can occur |

## Review Terms

| Term | Definition |
|------|------------|
| **Peer Review** | Evaluation by subject matter experts |
| **Study Section** | NIH peer review group |
| **Panel** | NSF review group |
| **Score** | Numerical rating from reviewers |
| **Percentile** | Ranking relative to other applications |
| **Fundable Range** | Score threshold for likely funding |
| **Just-in-Time** | Additional materials requested before funding decision |
| **Program Officer** | Funder staff managing the grant program |
| **IRB** | Institutional Review Board - human subjects oversight |
| **IACUC** | Institutional Animal Care and Use Committee |

## Organizational Terms

| Term | Definition |
|------|------------|
| **PI** | Principal Investigator - project lead |
| **Co-PI** | Co-Principal Investigator |
| **Sponsored Programs** | Office managing grants at an institution |
| **OSP** | Office of Sponsored Programs |
| **AOR** | Authorized Organizational Representative |
| **Fiscal Sponsor** | Organization that manages grants for another entity |
| **Consortium** | Group of organizations collaborating on a project |
| **Prime** | Lead organization in a multi-organization grant |
| **Sub** | Partner organization receiving subaward |

Domain knowledge for research funding landscape analysis, budget verification, and strategic funding decisions.

This file complements existing domain context:

## Research Funding Lifecycle

### Pre-Award Phase

| Stage | Activities | Key Players |
|-------|-----------|-------------|
| **Opportunity Identification** | Monitor FOAs, scan funder priorities, assess fit | PI, Grants Office |
| **Internal Review** | Limited submissions, institutional endorsement | Department, Provost |
| **Proposal Development** | Write narrative, develop budget, gather support | PI, Co-PIs, OSP |
| **Submission** | Route through institutional systems, verify compliance | OSP, AOR |
| **Review** | Peer review, study section scoring, program officer evaluation | External reviewers |
| **Award Negotiation** | Terms and conditions, budget adjustments, start date | OSP, Program Officer |

### Award Phase

| Stage | Activities | Key Players |
|-------|-----------|-------------|
| **Setup** | Account creation, hire personnel, order equipment | PI, Grants Accounting |
| **Execution** | Conduct research, manage team, track spending | PI, Lab Manager |
| **Reporting** | Progress reports, financial reports, compliance | PI, OSP |
| **Modifications** | No-cost extensions, budget revisions, scope changes | PI, Program Officer |

### Post-Award Phase

| Stage | Activities | Key Players |
|-------|-----------|-------------|
| **Closeout** | Final reports, equipment disposition, record retention | PI, OSP |
| **Audit** | Financial audit, effort verification, compliance review | Auditors, OSP |
| **Renewal/Resubmission** | Competitive renewal, new proposal development | PI |

## Foundation Funding

### Private Foundations

| Type | Examples | Typical Range | Application Process |
|------|---------|---------------|-------------------|
| **Large national** | Ford, MacArthur, Kresge | $100K-$2M | LOI then full proposal |
| **Science-focused** | Sloan, Simons, Moore | $50K-$1M | Invited or open |
| **Health-focused** | Gates, RWJF, Commonwealth | $100K-$5M | RFP or LOI |
| **AI/Tech safety** | Open Phil, SFF, LTFF | $50K-$2M | Application or referral |

### Community Foundations

- Typically smaller awards ($5K-$50K)
- Local/regional focus
- Simpler application process
- Often rolling deadlines
- Good for pilot projects and community-engaged research

### Corporate Foundations

- Aligned with corporate strategic interests
- May include in-kind support (equipment, data access)
- Shorter timelines than government grants
- May require branded deliverables or publicity

## Cost-Effectiveness Analysis Frameworks

### Cost per DALY (Disability-Adjusted Life Year)

```
Cost-Effectiveness Ratio = Total Program Cost / DALYs Averted

Thresholds (WHO guidelines):
- Highly cost-effective: < 1x GDP per capita per DALY
- Cost-effective: 1-3x GDP per capita per DALY
- Not cost-effective: > 3x GDP per capita per DALY
```

### ICER (Incremental Cost-Effectiveness Ratio)

```
ICER = (Cost_new - Cost_comparator) / (Effect_new - Effect_comparator)

US thresholds:
- Generally accepted: < $50,000/QALY
- Often accepted: $50,000-$150,000/QALY
- Rarely accepted: > $150,000/QALY
```

### Budget Impact Analysis

```
Budget Impact = (Population x Uptake x Unit Cost_new) - (Population x Current Treatment x Unit Cost_current)

Required components:
- Target population size and growth
- Expected uptake/adoption rate
- Per-unit costs of intervention
- Current standard of care costs
- Time horizon (typically 3-5 years)
```

## F&A (Indirect Cost) Rate Structures

### Rate Types

| Rate Category | Typical Range | Applied To |
|---------------|---------------|-----------|
| **On-campus research** | 40-65% | MTDC base |
| **Off-campus research** | 20-26% | MTDC base |
| **Instruction** | 40-60% | MTDC base |
| **Other sponsored** | 20-40% | MTDC base |

### MTDC (Modified Total Direct Costs) Base

Items **included** in MTDC:
- Salaries and wages
- Fringe benefits
- Materials and supplies
- Travel
- First $25,000 of each subaward

Items **excluded** from MTDC:
- Equipment (>$5,000)
- Patient care costs
- Tuition remission
- Subaward amounts over $25,000
- Rental costs of off-site facilities
- Scholarships and fellowships

### Negotiated Rate Agreement

- Negotiated with cognizant federal agency (usually HHS or ONR)
- Reviewed every 2-4 years
- Provisional rates used until final rate established
- Rate disputes resolved through established procedures

## Effort Reporting

### Person-Months Calculation

```
Person-Months = (% Effort / 100) x Number of Months in Appointment

Examples:
- 10% effort on 12-month appointment = 1.2 person-months
- 25% effort on 9-month appointment = 2.25 person-months
- 50% effort on summer (3 months) = 1.5 person-months
```

### Salary Cap Calculations

**NIH Salary Cap** (Executive Level II):
```
Capped Salary = min(Actual Salary, NIH Cap)
Effort Cost = Capped Salary x % Effort
Fringe = Effort Cost x Fringe Rate

Example (NIH cap $221,900, actual salary $250,000, 20% effort):
  Capped amount = $221,900 x 0.20 = $44,380
  Fringe (35%) = $44,380 x 0.35 = $15,533
  Total = $59,913
```

**NSF Two-Months Rule**:
- PI limited to 2 months of salary per year across all NSF awards
- Calculated as person-months, not calendar months
- Exceptions require prior written approval

### Cost-Sharing and Matching

| Type | Definition | Verification |
|------|-----------|-------------|
| **Mandatory** | Required by funder | Must be documented and auditable |
| **Voluntary committed** | Offered in proposal | Becomes binding if awarded |
| **Voluntary uncommitted** | Not in proposal | No reporting requirement |
| **In-kind** | Non-cash contributions | Must be valued at fair market |
| **Third-party** | From external entities | Requires documentation |

## Subaward vs Subcontract Distinctions

| Criterion | Subaward | Subcontract |
|-----------|----------|-------------|
| **Relationship** | Collaborative research partner | Vendor/service provider |
| **Scope** | Substantive programmatic work | Defined deliverable/service |
| **IP Rights** | Shared or negotiated | Typically retained by prime |
| **Budget Treatment** | First $25K in MTDC | Typically excluded from MTDC |
| **Reporting** | Progress reports required | Invoice-based |
| **F&A** | Sub applies own rate | No F&A (or included in price) |

## Budget Categories

### Standard Federal Budget Structure

| Category | What It Includes | Common Issues |
|----------|-----------------|---------------|
| **A. Personnel** | Salaries, wages, consultant costs | Salary cap compliance, effort allocation |
| **B. Fringe Benefits** | Health, retirement, FICA, workers' comp | Rate verification, composite vs actual |
| **C. Equipment** | Items >$5,000 with useful life >1 year | Excluded from F&A, justification needed |
| **D. Travel** | Domestic and foreign, per diem | Conference attendance, field work, collaborator visits |
| **E. Participant Support** | Stipends, travel, subsistence for participants | Excluded from F&A, cannot be re-budgeted |
| **F. Other Direct** | Materials, supplies, publication costs, fees | Itemize major categories |
| **G. Subawards** | Partner institution costs | Only first $25K per sub in MTDC |
| **H. Indirect (F&A)** | Facilities and administrative costs | Negotiated rate, correct base |

### Budget Justification Requirements

Each line item requires:
1. **What**: Specific item or service
2. **Why**: How it supports the project goals
3. **How much**: Calculation showing how cost was derived
4. **When**: Which project period(s)

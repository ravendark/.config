# Questions Slide - Q&A with Contact

## Slidev Template

```md
---
layout: center
class: questions
---

# Questions?

<div class="contact-info">

**{{presenter_name}}**

{{email}}

{{institution}}

{{social_or_website}}

</div>
```

## Content Slots

| Slot | Description | Required |
|------|-------------|----------|
| `presenter_name` | Presenter full name | Yes |
| `email` | Contact email | Yes |
| `institution` | Institution and department | No |
| `social_or_website` | Twitter/X handle, website, or ORCID | No |

## Usage Notes

- Keep the slide clean and uncluttered
- Use a large "Questions?" heading
- Include only essential contact information
- Consider adding a QR code linking to the paper or preprint
- This slide stays visible during the Q&A session

# Narrative Patterns

Writing patterns for grant narrative sections, including impact statements, problem framing, and persuasive structures.

## Overview

Effective grant narratives combine evidence with compelling storytelling while maintaining professional tone and precision.

## Impact Statement Pattern

### Formula

```
[Quantified outcome] + [Population affected] + [Timeframe] + [How measured]
```

### Examples

```
"This project will train 500 educators in trauma-informed practices
over 24 months, increasing student attendance by 15% as measured by
school records."

"The proposed research will reduce model training costs by 40% while
maintaining accuracy, enabling 10x more research teams to conduct
safety evaluations."
```

### Anti-patterns

```
WEAK: "This project will help many students."
WEAK: "We will make a significant impact."
WEAK: "Our work will advance the field."
```

## Problem Framing Pattern

### Structure

```markdown
## Statement of Need

**The Challenge**: [One sentence defining the problem]

[Evidence paragraph 1]: Scope and scale of the problem with data.
[Citation of credible source].

[Evidence paragraph 2]: Consequences of inaction. What happens if this
problem is not addressed? [Data on costs, harms, missed opportunities].

[Gap paragraph]: Current solutions are insufficient because [reasons].
Specifically, [gap 1], [gap 2], and [gap 3] remain unaddressed.

[Target population paragraph]: This problem particularly affects
[specific population]. In [location/context], [specific local data].

**The Opportunity**: [One sentence framing the solution opportunity]
```

### Evidence Hierarchy

1. **Peer-reviewed research**: Most credible
2. **Government statistics**: Reliable and citable
3. **Program evaluation data**: Shows real-world results
4. **Expert opinion**: Named, credentialed sources
5. **Anecdotal evidence**: Use sparingly, with caution

## SCAR Pattern (for Capacity)

### Structure

```
SITUATION: What context or challenge did you face?
CHALLENGE: What specific obstacle or goal?
ACTION: What did you do?
RESULT: What measurable outcome?
```

### Example

```
In 2023, our organization faced a critical shortage of trained
evaluators (SITUATION). We needed to double our evaluation capacity
within six months to meet client demand (CHALLENGE). We developed an
intensive training program and recruited 15 recent graduates from
partner universities (ACTION). Within eight months, we increased
evaluation capacity by 150% and reduced turnaround time by 40%
(RESULT).
```

## Theory of Change Narrative

### Pattern

```markdown
**Theory of Change**

We believe that [target population] face [problem] because of
[root causes].

Evidence suggests that [intervention type] can address these
causes by [mechanism]. Specifically, [citation or evidence].

Our project will therefore [specific activities], which will
directly address [root cause 1] and [root cause 2].

Based on similar interventions, we expect [expected outcomes]
within [timeframe]. Success indicators include [measurable
indicators].
```

## Active Voice Transformations

| Passive (Avoid) | Active (Use) |
|-----------------|--------------|
| "Participants will be trained..." | "We will train participants..." |
| "Data will be collected..." | "Research staff will collect data..." |
| "The program was developed..." | "Our team developed the program..." |
| "It is expected that..." | "We expect..." |
| "There is evidence that..." | "Research shows that..." |

## Transition Patterns

### Between Sections

```
"Having established the scope of this challenge, we now describe
our approach to addressing it."

"Building on these findings, our project will..."

"The methodology described above leads to the following budget
requirements..."
```

### Within Sections

```
"First... Second... Third..."
"Initially... Subsequently... Finally..."
"In addition to... Furthermore... Moreover..."
```

## Specificity Ladder

Move from vague to specific:

```
Level 1 (Vague): "We will train teachers."

Level 2 (Better): "We will train 50 teachers in evidence-based
practices."

Level 3 (Specific): "We will provide 40 hours of training in
trauma-informed classroom management to 50 elementary teachers
across 10 schools in Springfield District over 12 months."

Level 4 (Optimal): "We will provide 40 hours of training (20
in-person, 20 online) in trauma-informed classroom management,
based on the [Named Model], to 50 elementary teachers (grades
K-5) across 10 Title I schools in Springfield District, delivered
in four 10-hour modules between September 2024 and August 2025."
```

## Sentence-Level Patterns

### The "So What" Test

After each sentence, ask "So what?" Add significance:

```
BEFORE: "Our program served 200 participants last year."

AFTER: "Our program served 200 participants last year, exceeding
our target by 25% and demonstrating strong community demand for
these services."
```

### The Data Sandwich

```
[Claim] + [Data] + [Significance]

"Teacher retention is a critical challenge [claim]. The district
loses 23% of new teachers within three years [data]. This
turnover costs an estimated $500,000 annually and disrupts
student learning [significance]."
```

### Parallel Construction

For lists of activities, objectives, or outcomes:

```
GOOD:
The project will:
- Develop curriculum materials
- Train program facilitators
- Evaluate program outcomes
- Disseminate findings widely

POOR:
The project will:
- Development of curriculum
- Training facilitators
- We will evaluate outcomes
- Dissemination
```

## Word Choice Patterns

### Strong Verbs

| Weak | Strong |
|------|--------|
| will try to | will |
| hope to achieve | will achieve |
| may result in | will result in |
| help with | enable / support / strengthen |
| look at | analyze / examine / assess |
| do | implement / execute / conduct |

### Precise Quantifiers

| Vague | Precise |
|-------|---------|
| many | 347 / more than 300 |
| significant increase | 23% increase |
| most | 85% / 17 of 20 |
| frequently | weekly / 4 times per month |
| some | 12 / approximately 10-15 |

## Avoiding Common Problems

### Jargon Reduction

```
JARGON: "We will leverage synergistic partnerships to
operationalize a holistic, trauma-informed paradigm shift."

CLEAR: "We will work with three community partners to implement
trauma-informed practices across all program areas."
```

### Circular Logic

```
CIRCULAR: "This project is important because it addresses an
important issue."

CLEAR: "This project addresses a critical gap: no existing
program provides [specific service] to [specific population]
in [specific area]."
```

## Best Practices

1. **Write for skimmers**: Use headers, bullets, and bold text
2. **Lead with impact**: Put most important information first
3. **Be specific**: Vague claims weaken credibility
4. **Show evidence**: Back claims with data and citations
5. **Stay positive**: Frame challenges as opportunities
6. **Match funder voice**: Use their terminology and priorities
